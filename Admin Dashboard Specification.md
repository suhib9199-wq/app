# Admin Dashboard Specification

## Roles
- Admin
- Manager
- Staff

## Modules
- Overview Dashboard
- Leads Management
- Requests Management
- Appointments
- Companies
- Content Management
- AI Agent Scripts
- Reports
- Settings

## Permissions
Admin: Full access.
Manager: View reports, assign leads, manage team tasks.
Staff: Handle assigned leads and requests.

## Dashboard Widgets
- Total Leads
- New Leads
- Conversion Rate
- Requests by Service
- Appointments Calendar
- Recent Activity
