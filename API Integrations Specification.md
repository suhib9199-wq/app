# API Integrations Specification

## Integrations
- CRM API
- WhatsApp Link / WhatsApp Business API readiness
- Email SMTP / Transactional Mail
- Google Analytics
- Search Console
- Calendar / Appointment System
- AI Agent Endpoint

## API Standards
- JSON request/response
- Authentication via token
- Rate limiting
- Error logging
- Retry mechanism

## Core Endpoints
- POST /api/leads
- GET /api/leads
- PATCH /api/leads/{id}
- POST /api/appointments
- POST /api/contact
- POST /api/ai/chat
